# Escopo do MVP

## Objetivo do MVP

Entregar a menor versão útil do ERP para que a empresa consiga operar o NossoZelo com controle financeiro, fiscal e operacional desde o início.

## Módulos do MVP

### 1. Dashboard

Deve exibir:

- receita do mês;
- despesas do mês;
- saldo previsto;
- lucro ou prejuízo estimado;
- contas a pagar vencendo;
- contas a receber;
- assinaturas ativas;
- assinaturas atrasadas;
- chamados abertos;
- obrigações MEI pendentes;
- tarefas urgentes.

### 2. Financeiro básico

Funcionalidades:

- criar lançamento financeiro;
- classificar como receita ou despesa;
- definir categoria;
- definir data de competência;
- definir data de pagamento;
- definir valor;
- definir status;
- anexar comprovante;
- filtrar por mês;
- filtrar por categoria;
- ver total de entradas e saídas.

### 3. Contas a pagar

Funcionalidades:

- cadastrar despesa futura;
- definir vencimento;
- definir recorrência;
- marcar como paga;
- anexar comprovante;
- gerar alerta de vencimento;
- relacionar com categoria financeira.

### 4. Contas a receber

Funcionalidades:

- registrar valor a receber;
- relacionar com assinatura, cliente ou prestador;
- definir vencimento;
- marcar como recebido;
- registrar forma de pagamento;
- registrar taxa;
- registrar valor líquido.

### 5. Obrigações MEI

Funcionalidades:

- cadastrar DAS mensal;
- definir vencimento;
- registrar pagamento;
- anexar comprovante;
- registrar declaração anual;
- alertar pendências.

### 6. Notas fiscais

Funcionalidades:

- registrar nota fiscal emitida manualmente;
- informar tomador;
- informar documento do tomador;
- informar valor;
- informar data;
- informar número da NFS-e;
- anexar PDF/XML;
- marcar status.

### 7. Assinaturas

Funcionalidades:

- listar assinaturas do marketplace;
- ver status;
- ver vencimento;
- ver histórico de pagamento;
- ver inadimplência;
- registrar observações administrativas.

### 8. Clientes e prestadores

Funcionalidades:

- listar usuários do marketplace;
- filtrar por tipo;
- ver status;
- ver histórico básico;
- relacionar chamados;
- relacionar assinaturas;
- visualizar pendências.

### 9. Suporte

Funcionalidades:

- criar chamado;
- relacionar chamado a usuário ou prestador;
- definir tipo;
- definir prioridade;
- definir status;
- registrar histórico;
- encerrar chamado.

### 10. Tarefas internas

Funcionalidades:

- criar tarefa;
- definir responsável;
- definir prioridade;
- definir prazo;
- definir módulo relacionado;
- mudar status;
- registrar conclusão.

### 11. Auditoria

Funcionalidades:

- registrar ações administrativas relevantes;
- registrar usuário interno que executou;
- registrar data/hora;
- registrar entidade afetada;
- registrar motivo, quando aplicável.

## Critérios de aceite do MVP

O MVP será considerado pronto quando permitir:

- cadastrar receitas e despesas;
- consultar resultado financeiro mensal;
- controlar contas a pagar;
- controlar contas a receber;
- registrar pagamento do DAS;
- registrar notas fiscais emitidas manualmente;
- ver prestadores e clientes do marketplace;
- acompanhar assinaturas;
- criar e acompanhar chamados;
- criar e acompanhar tarefas internas;
- auditar ações administrativas críticas.
