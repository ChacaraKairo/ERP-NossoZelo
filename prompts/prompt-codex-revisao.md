# Prompt Codex — Revisão

```text
Você está revisando o projeto ERP-NossoZelo.

Objetivo:
Revisar documentação, código e estrutura para garantir que o ERP atende ao escopo do MVP.

Verifique:

1. Se o README está coerente.
2. Se todos os documentos em docs/ foram considerados.
3. Se o módulo de Serviços Contratados foi tratado como obrigatório.
4. Se o financeiro permite controlar receitas, despesas, contas a pagar e receber.
5. Se obrigações MEI e notas fiscais manuais estão contempladas.
6. Se há auditoria para ações críticas.
7. Se permissões foram respeitadas.
8. Se não há segredos no código.
9. Se `.env` está ignorado.
10. Se migrations funcionam em banco limpo.
11. Se lint, testes e build passam.

Dê uma resposta objetiva com:

- problemas encontrados;
- riscos;
- correções sugeridas;
- arquivos afetados;
- próximos passos.

Não faça alterações sem explicar o motivo.
```
