# Prompt Codex — Planejamento

Use este prompt no Codex para revisar o planejamento do ERP.

```text
Você está trabalhando no repositório ERP-NossoZelo.

Objetivo:
Criar um ERP interno para a empresa que opera o marketplace NossoZelo.

Antes de implementar qualquer código:

1. Leia o README.md.
2. Leia todos os documentos em docs/.
3. Entenda o contexto da empresa MEI.
4. Entenda que o ERP deve durar pelo menos 2 anos.
5. Entenda que o foco é baixo orçamento, simplicidade e operação real.
6. Não proponha módulos exagerados fora do escopo.
7. Priorize:
   - financeiro;
   - serviços contratados;
   - obrigações MEI;
   - notas fiscais manuais;
   - clientes/prestadores;
   - assinaturas;
   - suporte;
   - tarefas;
   - auditoria.

Depois, gere um plano de implementação em etapas pequenas, com:

- arquivos a criar;
- ordem de implementação;
- critérios de aceite;
- riscos;
- comandos de teste.

Não escreva código antes de apresentar o plano.
```
