# Prompt Codex — Frontend

```text
Você está implementando o frontend do ERP-NossoZelo.

Leia antes:
- README.md
- docs/03-escopo-mvp.md
- docs/07-telas-e-fluxos.md
- docs/08-perfis-e-permissoes.md
- docs/15-servicos-contratados.md

Objetivo:
Criar a interface administrativa do ERP.

Layout obrigatório:
- aplicação web administrativa;
- menu lateral;
- área de conteúdo principal;
- dashboard inicial;
- telas em português;
- foco em clareza e produtividade.

Telas prioritárias:
1. Dashboard.
2. Financeiro.
3. Contas a pagar.
4. Contas a receber.
5. Serviços contratados.
6. Fiscal/MEI.
7. Notas fiscais.
8. Marketplace.
9. Assinaturas.
10. Suporte.
11. Tarefas.
12. Auditoria.

Regras de UX:
- não esconder informações financeiras importantes;
- destacar contas vencidas;
- destacar serviços críticos;
- destacar obrigações MEI próximas do vencimento;
- exibir filtros claros;
- permitir criação rápida de registros;
- evitar telas poluídas;
- usar tabelas quando houver muitos registros;
- usar cards no dashboard.

Critérios de aceite:
- lint passando;
- build passando;
- navegação funcionando;
- telas principais acessíveis;
- formulários com validação;
- estados vazios tratados;
- feedback de erro e sucesso.
```
